Fun Game!

Fun Game! has a long story.

So originally it started as just a test to see what I could build in 1 hour. And after that one hour I had a simple little game,
and I kind of forgot about it. Then one day I said, "Hey, let's remake this game but for the Game Boy." So that's what I did.
I remade Fun Game! for the Game Boy, and I had so much fun making it! After reading a few comments on the itch.io page,
I decided to make Fun Game! a complete game! So that's what I did!
